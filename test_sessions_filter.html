Sessions Filter Functionality Test

To test the sessions filtering functionality:

  1. Open http://localhost:5173
  2. Navigate to the Coach Panel > Sessions page
  3. You should see the filter buttons: "All Sessions", "Upcoming", "Today", "Finished"
  4. Each filter button shows a count of sessions in that category
  5. Click each filter to see the sessions filtered accordingly:
    • All Sessions: Shows all 6 sessions
    • Upcoming: Shows future sessions (including today)
    • Today: Shows only today's sessions
    • Finished: Shows past sessions (excluding today)
  6. The status badges on each session show "Upcoming" or "Completed" based on the date

Expected Results:

  • Filter buttons are visually distinct when active (blue background)
  • Session counts are displayed next to each filter
  • Sessions are properly filtered based on their dates
  • Empty state message changes based on the active filter

Mock Data Included:

  • 2 upcoming sessions (2-3 days from now)
  • 1 today's session
  • 2 finished sessions (1 and 3 days ago)
  • 1 tomorrow's session
